GoboLinux 014 - Known issues

Updated packages

The following updated packages were released fixing problems detected in the GoboLinux 014 release. After installing the Live CD, you should install these packages using the InstallPackage command. For example, to upgrade Xorg, just run "InstallPackage xorg" from the command line and it will download and install the latest version.

  • Xorg 7.2-r6 - fixes LibXCB assertion errors.

Updated recipes

The following updated recipes were released fixing problems detected in the GoboLinux 014 release. Packages built from these should come up shortly. In the mean time, you can use the Compile tool to build the upgraded versions.

  • DBus 1.1.2-r3 - fixes system id problem.

AM2 problems with SquashFS

There seems to be a bug on SquashFS affecting Dual core AM2 AMD64 systems. These system boots after supplying the following command at the CD bootloader prompt: 

/isolinux/kernel initrd=initrd init=/linuxrc ramdisk_size=4096 root=/dev/ram0 video=vesafb:off noapic nolapic nosmp apic=off Boot=LiveCD max_loop=32

This problem does not affect the installed system, which works with SMP/HT/AM2 fine.

Other issues

The LiveCD is known to have problems booting on the VirtualBox OSE emulator. This has been fixed in the in-development version of the distribution.
